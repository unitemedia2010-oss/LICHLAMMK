# V9 - Premium Mobile Calendar UI

## Đã tối ưu

- Giao diện liquid glass / iOS-like với nền blur, lớp kính, gold glow #ffd700.
- Thêm watermark UNITE siêu lớn dưới nền để tạo chiều sâu.
- Thêm icon cho các card chỉ số.
- Mobile không còn chuyển lịch tháng thành danh sách dài nữa.
- Mobile giữ dạng bảng lịch tháng 7 cột.
- Bấm ngày trên lịch nhân sự để mở popup chi tiết ngày.
- Bấm/giữ ngày trên lịch admin để mở popup chi tiết nhân sự đăng ký.
- Popup chi tiết dùng nền mờ, hiệu ứng bay lên và glass blur.
