# V13 Enterprise Responsive

- Chỉ dùng `style.css` và `v13-enterprise-responsive.css` để tránh xung đột nhiều lớp CSS.
- Thiết kế sáng, tối giản, tập trung typography và hệ thống khoảng cách.
- Mobile giữ bảng lịch 7 cột; bấm ngày mở bottom sheet chi tiết.
- Ô hôm nay chỉ viền vàng `#FFD700`, không thêm chữ.
- Tối ưu safe-area, `100dvh`, font input 16px trên iOS, touch target tối thiểu 44px.
- Không cần chạy SQL mới.
