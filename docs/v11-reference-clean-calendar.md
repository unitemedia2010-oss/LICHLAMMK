# V11 Reference Clean Calendar

Bản này chỉnh giao diện theo sườn React tham khảo:

- Màu chủ đạo: nâu / kem / trắng sạch.
- Bỏ gold glow mạnh, bỏ emoji icon lớn.
- Card thống kê dùng icon nhỏ trong ô bo góc.
- Mobile calendar giữ grid 7 cột, không xổ dọc theo ngày.
- Ngày hôm nay chỉ có viền vàng, không hiện chữ “Hôm nay”.
- Bấm vào ngày vẫn mở popup chi tiết như bản trước.

Không cần chạy thêm SQL nếu đã chạy các bản v8/v9/v10.
